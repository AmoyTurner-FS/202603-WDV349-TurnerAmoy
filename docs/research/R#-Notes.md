## GitHub Feature Branch Workflow

Feature Branch Workflow is a way developers organize their work when using Git. Instead of working directly on the main branch, developers create separate branches for different tasks or features. This helps keep the main code stable while new work is being developed.

## Branch Structure 

In this course we are using multiple branches to manage development.

Feature branches are used for new work or tasks.  
The dev branch collects the latest development changes.  
The stage branch is used to test work before it is finalized.
