## Project: CarFinder

CarFinder is a web application that allows users to search for vehicles by year, make, and model. The goal is to create a simple, clean, and user friendly interface for quickly exploring car data without unnecessary complexity.

This project will demonstrate full stack development by integrating a front end interface, API data, and dynamic filtering functionality.
